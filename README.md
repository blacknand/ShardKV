# ShardKV
A fast distributed, self-healing key-value store designed for fault-tolerance and scalability.
## Installation
```bash
git clone git@github.com:blacknand/ShardKV.git
cd ShardKV
mkdir build && cd build
cmake ..
make
```
## Usage
```bash
./shardkv
```
## Requirements
- `GCC` or `clang`
- `CMake`
- GoogleTest
